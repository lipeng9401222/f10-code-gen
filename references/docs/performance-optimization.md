# 性能优化

> epoint Vue 前端性能优化指南

## 表格性能优化

对于大数据量表格，推荐使用 `EpDataGrid` 或配置虚拟滚动。

详细文档：[components-eui-core-ep-datagrid.md](./ep-components/components-eui-core-ep-datagrid.md)

## 虚拟滚动选择器

当下拉选项过多时，请使用虚拟滚动选择器。

详细文档：[components-select-virtual.md](./components/components-select-virtual.md)

- 路由懒加载
- 组件按需引入 (框架已默认支持)
