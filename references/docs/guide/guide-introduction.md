## 介绍

eui-components 是一套基于 Vue.js （v3）的 UI 组件库，适用于 PC 端，以 Epoint UI 的规范实现。

## 浏览器兼容性

eui-components 支持以下浏览器：

| ![IE](./images/edge_32x32.png) | ![Firefox](./images/firefox_32x32.png) | ![Chrome](./images/chrome_32x32.png) | ![Safari](./images/safari_32x32.png) |
| --- | --- | --- | --- |
| Edge ≥ 88 | Firefox ≥ 84 | Chrome ≥ 88 | Safari ≥ 14.1 |

> **⚠️ 警告**
>
> - 因 Vue3 不兼容 IE，所以 Epoint UI Components 也不支持在 IE 下使用。
