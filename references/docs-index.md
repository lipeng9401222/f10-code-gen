## 使用建议（给 AI）

1. **不要凭记忆**写 API。任何"我记得 ep-data-grid 有 xxx 属性"的话，先查对应 `.md`
2. **优先 frontpage.md + dialog-interaction.md**：F10 强制规范的最高优先级
3. **数据模型**：先看 `api/api-hooks-data-fetch-hooks.md`（写得最详细）
4. **组件**：先看 `components-reference.md` 速查，深入再看具体 `components-<name>.md`
5. **页面**：优先 `examples-index.md`（不在本文件而是另一个）
